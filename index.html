JSON Lines: Simple. Like JSON.

This page describes the JSON Lines text format. JSON Lines is a convenient format for storing structured data that may be processed one record at a time. It works well with unix-style scripts shell pipelines.

JSON Lines files may be saved with the file extension .jl. Stream compressors like gzip or bzip2 are recommended for saving space, resulting in .jl.gz or .jl.bz2 files.
